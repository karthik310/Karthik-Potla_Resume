# Resume Website (Simple)

This is a minimal, responsive resume website you can customize and show in interviews.

Files added:
- `index.html` — main page
- `styles.css` — layout and print styles
- `script.js` — small UI enhancements

To preview locally, run a simple HTTP server from the project root and open http://localhost:8000:

```bash
python3 -m http.server 8000
```

Replace `resume.pdf` with your real PDF to enable the download button.

Customize the content in `index.html` and the styles in `styles.css`.
